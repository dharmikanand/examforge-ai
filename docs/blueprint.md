# **App Name**: ExamForge AI

## Core Features:

- Multimodal Input: Accepts text, images (JPG, PNG), and PDF uploads; processes different formats, extracts and converts content to feed into AI engine.
- AI Exam Strategist: Processes inputs via AI models and generates outputs based on one of four modes.
- 5-Minute Survival Mode: Provides ultra-condensed revisions, key formulas, important definitions, and critical theorems.
- Exam Weaponizer Mode: Generates probable exam questions, predicted weightage, important derivations, and strategic study suggestions; tool uses AI reasoning.
- Concept Trap Detector: Highlights common mistakes, misconceptions, trick-based questions, and frequently confused concepts; tool uses AI reasoning.
- MCQ Generator: Generates multiple-choice questions with mixed difficulty, answers, and short explanations.
- Dynamic UI Cards: Displays outputs in structured, collapsible UI cards with animated reveals and a copy-to-clipboard option.

## Style Guidelines:

- Primary color: Deep blue (#293B5F), evoking intelligence and focus.
- Background color: Light gray (#D2D6DD), providing a clean, distraction-free environment.
- Accent color: Bright orange (#E87A00), used for call-to-action elements and important highlights.
- Body and headline font: 'Inter', a sans-serif font for modern legibility and clean information presentation.
- Code font: 'Source Code Pro' for code snippets.
- Use clear and consistent icons to represent different types of content and functionalities.
- Employ a clean and structured layout to ensure content is easily scannable and digestible.
- Use subtle animations for feedback and to indicate progress during file processing and intelligence generation.
